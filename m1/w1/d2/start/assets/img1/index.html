Come creare un template per i post

di Mario Rossi - pubblicato il 25-10-2020

La prima cosa da fare è suddividere in blocchi logici la pagina, identificare le parti comuni da applicare a tutte le pagine.

Le varie aree saranno quindi suddivise in:

  • Contenuti di intestazione del tag <Header>
  • Contenuti centrali variabili dal tag <Main>
  • Contenuti di fine pagina comuni, da inserire nel tag <footer>
  • Formattazione del codice

Il testo dell'articolo dovrà essere formattato per identificare

  • Progettare su carta il markup finale, assegnando ad ogni specifico appropriato
  • Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  • Applicare gli stili CSS
  • L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressoché totale
